建站小记

2025/12/4 教程 Hexo

简单记录一下建站参考的一些文章

我的网站部署采用了hexo + github pages的形式。

1.部署hexo

安装教程参考了这个博客，也可以通过官方文档部署。
我的环境是 Windows，所以写了 .bat 自动化部署脚本

@echo off
hexo clean && hexo generate && hexo deploy
echo 网站已更新
pause

2.定制主题

可以去 Hexo主题中选择自己喜欢的主题，将主题下载到 theme 文件夹下，再根据主题的文档进行自己的配置。我使用的是particlex
